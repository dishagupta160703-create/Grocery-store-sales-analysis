## Grocery Store Sales Analysis 📊


##  Project Overview :

*  This project focuses on analyzing Grocery Store sales data to understand sales performance, product demand, and profit trends. The goal of this analysis is to extract useful business insights that can help improve decision-making.

*  The project uses Excel for data cleaning, SQL for data analysis, and Power BI for creating interactive dashboards.


##  Objectives :

*  Analyze sales performance across different product and categories

*  Identify top-selling products

*  Understand monthly sales trends

*  Analyse order distriution across payment mode

*  Identify sales by city

*  Analyse Orders distribution by member type


##  Tools & Technologies :

*  Microsoft Excel – Data cleaning and preprocessing

*  SQL – Data querying and analysis

*  Power BI – Dashboard and data visualization


##  Dataset Information :

The dataset contains the following fields:

*  Order ID

*  Order Date

*  Store

*  Product

*  Category

*  Unit Price

*  Quantity

*  City
  
*  Payment Mode

*  Customer type

*  Total Revenue


##  Data Analysis Process :


*  Data Cleaning using Excel

*  Data Analysis using SQL queries

*  Data Visualization using Power BI


##  Key Insights :


*  Certain product and categories generate higher sales.

*  Sales trends vary across different months.

*  Some city contribute more to overall revenue.

*  Order distribttion across payment mode


##  Dashboard :

The Power BI dashboard provides visual insights including:

*  Sales by Category

*  Monthly Sales Trend

*  Distribution of orders by Payment mode

*  Top 5 Selling Products

*  Sales by city

* Order distribution by member type
